# Reaction Timer

Simple reaction time tester built with plain HTML/CSS/JavaScript.

- **How to run:** Open `reaction-timer/index.html` in your browser (no server required).
- **Goal:** Click as fast as possible after the screen turns green. Early clicks count as faults.
- **Features:** Records last attempt and best time (stored in `localStorage`). Reset best time with the button.
